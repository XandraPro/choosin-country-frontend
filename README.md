# 🎨 Choosin Country Frontend

Frontend application for Choosin Country.  
Built with React, it allows users to discover country songs, save favourites, add comments and explore live events.

---

## 🚀 Live App

[https://TU-FRONTEND-URL.vercel.app](https://TU-FRONTEND-URL.vercel.app)

---

## 🛠 Tech Stack

- React
- Axios
- CSS
- Vite

---

## ⚙️ Installation

Clone the repository and navigate to the backend directory:

cd choosin-country-frontend

## ⚙️ Install dependencies

npm install

## Run the server

npm run dev

## The server will run at

<http://localhost:5173>

## Make sure the backend is running or deployed

<https://choosin-country-backend.onrender.com>

---

## Features

- User authentication (login/register)
- Search Country songs
- Save songs to personal collection
- Add and edit comments
- View trending songs and ranking
- Browse country concerts and festivals
- Filter events by country, artist, city and type
